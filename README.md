# team-project-git-Group-2-
Практична робота 8
Командна розробка з використанням Git та GitHub

Мета: Навчитися ефективно використовувати Git та GitHub для спільної роботи в команді з двох осіб:
Планування роботи через гілки (feature, dev)
Створення Pull Requests та проведення взаємного рев’ю коду
Набуття навичок вирішення конфліктів злиття (Merge Conflicts)
Інструменти: Термінал (термінал IDE), Git (локально встановлений), GitHub акаунт
12345 вказівки
Робота виконується в групах по 2 студенти
Студент A - власник репозиторію, Студент B - співробітник
Коміти: Повідомлення мають бути змістовними. Використовуйте префікси: feat: (нова функція), fix: (виправлення), docs: (документація)
Всі зміни вносяться через Pull Request
